## Steps to setup

1. Create a virtual environment
   nodeenv --node=14.15.0 env
2. Start the env
   cd .\env\Scripts\
   .\Activate.ps1
3. Install node modules
   //Back in the app directory
   npm install
4. Start npm
   npm start
